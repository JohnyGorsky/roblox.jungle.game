# P2 — Role stations (swappable)

**Source:** Job 001 roadmap. **Depends:** P1.

Swappable stations + seat/station interaction; works for ~1–6 players (limited stations + deck
defenders for extras). Mobile-first controls per role.

## Roles + station bonuses (decided 2026-07-18 — see GAME.md "Player roles")
The stations already exist (drive / gun / fuel / repair); this job adds the **role assignment + the
efficiency bonus each specialist gets while manning their station**:
- **Driver** (one) — already built.
- **Gunner** (one) — mounted gun already built (#014).
- **Repair** (one) — **repairs faster** at the repair station.
- **Refuel / engineer** (one) — **refuels faster AND the boat burns less fuel** while manned.
- **Helpers / fighters** (the rest) — flexible; gun/sword, defend, scavenge, haul, revive.

Scope: how a player "takes" a role (sit / claim), enforce one-per-specialist, apply the bonus while
manned, a small role UI, swappability mid-run. Ties into the weapon loadout (Planned) for helpers.
